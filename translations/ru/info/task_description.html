Более шести лет назад, в декабре 1989, я искал хобби-проект по программированию, который бы занял у меня неделю до Рождества. Мой офис (государственная исследовательская лаборатория в Амстердаме) был закрыт, но у меня был домашний компьютер. Я решил написать интерпретатор для нового скриптового языка. Я думал о потомке ABC, чтобы обратить внимание Unix/C хакеров. Я выбрал Python в качестве рабочего названия проекта, будучи в не очень хорошем настроении (и большим поклонником Летающего цирка Монти Майтона).

Сегодня я могу с уверенностью сказать, что Python изменил мою жизнь. Я переехал на другой континент. Я провожу мои рабочие дни, разрабатывая большие системы на Python, когда я не программирую на Python или отвечаю на Python-подобные сообщения. Есть футболки Python, семинары, списки расылок, телеконференции, и теперь книга. Честно говоря, мое единственное желание иметь свою фотографию на первой странице New York Times.

-- Гвидо ван Россум, предисловие к "Programming Python", Рестон, Верджиния, Май 1996

Эта миссия имеет простое решение. У вас есть функция под названием "i_love_python" которая будет только возвращать фразу - "I love Python!"

Давайте напишем сочинение в Python коде, который будет объяснять, почему вы любите Python (если вы его не любите, тогда мы сделаем дополнительную миссию специально для ненавистников). За публикацию решения по умолчанию можно будет заработать только 0 очков, так как цель состоит в том, чтобы заработать очки через голоса для вашего кода-эссе.

Входные данные: Ничего.

Выходные данные: Строка "I love Python!".

Примеры: 

i_love_python() == "I love Python!"

Как это используется: Эта задача - это ваше сочинение в коде и программировании, как литература.
